# 量子计算入门

## 第一章：什么是量子计算

量子计算是一种利用量子力学原理进行信息处理的计算范式。与传统计算机使用比特（0或1）不同，量子计算机使用量子比特（qubit），它可以同时处于0和1的叠加态。

### 量子比特

量子比特是量子计算的基本单位。一个量子比特可以表示为一个二维希尔伯特空间中的向量：

```
|ψ⟩ = α|0⟩ + β|1⟩
```

其中 |α|² + |β|² = 1。测量时会以概率 |α|² 得到 |0⟩，以概率 |β|² 得到 |1⟩。

### 量子叠加

叠加原理是量子力学的核心特征之一。在经典世界中，一个硬币要么正面朝上要么反面朝上；但在量子世界中，一个量子比特可以同时"既是0又是1"，直到被测量为止。

## 第二章：量子纠缠

量子纠缠是另一个重要的量子现象。当两个或多个量子比特处于纠缠态时，它们的状态无法被独立描述，只能作为一个整体来理解。

爱因斯坦曾将量子纠缠称为"幽灵般的超距作用"，因为它允许两个相距遥远的粒子瞬间影响彼此的状态。

### 贝尔不等式

约翰·贝尔在1964年提出了贝尔不等式，为实验验证量子纠缠提供了理论基础。后来的实验证明量子力学确实违反了贝尔不等式，证实了量子纠缠的真实性。

## 第三章：姚期智与量子计算

姚期智（Andrew Chi-Chih Yao）是华裔计算机科学家，2000年图灵奖获得者。他在量子计算领域做出了多项开创性贡献。

姚期智在1993年发表的论文中首次提出了量子通信复杂度的概念，证明了在某些通信场景下，量子计算相比经典计算具有指数级别的优势。这一发现为后来量子计算理论的发展奠定了基础。

此外，姚期智在密码学领域的贡献——"姚氏百万富翁问题"——也与量子安全多方计算密切相关。
